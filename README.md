# Vite React Calculator

Ovo je kalkulator aplikacija napravljena pomoću React-a, TypeScript-a i Vite-a.

Aplikacija omogućava korisniku da unese dva broja i izvrši osnovne aritmetičke operacije:

- sabiranje
- oduzimanje
- množenje
- deljenje

Rezultat se prikazuje nakon klika na izabranu operaciju.
